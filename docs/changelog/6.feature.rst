Add a WHATWG-conformant HTML tokenizer: :func:`turbohtml.tokenize` for whole strings, the streaming
:class:`turbohtml.Tokenizer`, and the :class:`turbohtml.Token` / :class:`turbohtml.TokenType` types. The C state machine
is validated against the html5lib-tests tokenizer conformance suite and bulk-scans text runs the way html5ever does.
