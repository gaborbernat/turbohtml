Minify CSS with :func:`turbohtml.clean.minify_css` and :func:`turbohtml.clean.minify_css_inline`, a value-safe minifier
that rewrites every value to its shortest spec-equivalent form. It produces the smallest output of any round-trip-safe
minifier on PyPI, replacing `rcssmin <https://github.com/ndparker/rcssmin>`__ and `csscompressor
<https://github.com/sprymix/csscompressor>`__, which have migration guides - by :user:`gaborbernat`.
